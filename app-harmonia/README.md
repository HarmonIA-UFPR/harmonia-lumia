# Projeto HarmonIA

## Sumário Executivo

O **HarmonIA** é um projeto de pesquisa e desenvolvimento que busca integrar ferramentas de Processamento de Linguagem Natural (PLN) em uma plataforma interativa, 
voltada para curadoria colaborativa, avaliação de ferramentas de IA e suporte à ciência aberta.  

O projeto é conduzido por diferentes equipes de bolsistas de iniciação científica e voluntários, com responsabilidades complementares, 
garantindo o avanço das etapas de pesquisa, desenvolvimento técnico, design e usabilidade.  

---

## Objetivos Principais

- Pesquisar, avaliar e implementar ferramentas de PLN.
- Desenvolver banco de dados e frontend da plataforma.
- Criar um sistema de curadoria colaborativa.
- Projetar identidade visual e interface gráfica.
- Implementar funcionalidades de feedback e avaliação de usuários.
- Testar e validar o protótipo com foco em usabilidade.
